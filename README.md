# Лабораторная работа №1: "Введение в нейронные сети"

Данный репозиторий содержит файлы выполненной лабораторной работы №1

## Содержание

- *lite* – задание уровня Lite. Цель – создать таблицу точности из 18 экспериментов, основанных на переборе гиперпараметров на наборе геометрических фигур;
- *lite_pytorch* – задание уровня Lite, выполненное с помощью библиотеки pytorch;
- *pro* – задание уровня Pro. Цель – создать и обучить моделя для распознавания двух категорий изображений;
- *ultra_pro* – задание уровна Ultra Pro. Цель – создать и обучить модель для распознавания цифры, написанной от руки.
